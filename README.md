# Kiwi Prome Feed

Public PromeRotation community-plugin feed for KiwiKing releases.

## Add this source in PromeRotation

```text
https://raw.githubusercontent.com/jujubui/kiwi-prome-feed/main/catalog/mdraw/index.json
```

The feed contains separate manifests for:

- Dalamud API 15 / current NA-CN PromeRotation
- Dalamud API 12 / TC PromeRotation

PromeRotation automatically filters out packages for the other API level.
